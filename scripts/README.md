# Scripts

Standalone Python scripts for pipeline execution outside Jupyter.

| Script | Purpose |
|---|---|
| `download_data.py` | *(placeholder)* Download raw datasets from source links |
| `run_preprocessing.py` | End-to-end data cleaning and feature engineering |
| `train.py` | *(Task 2)* Train and save models |
| `evaluate.py` | *(Task 2)* Generate evaluation metrics and plots |

## Usage

```bash
# From repo root
python scripts/run_preprocessing.py
```
